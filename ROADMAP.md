# Roadmap

Nexo is currently in the planning and pre-alpha stage. The roadmap is intentionally outcome-focused and may change as the product is validated.

## Phase 0 — Product definition

- [x] Define the personal and household organization concept
- [x] Establish privacy and collaboration principles
- [x] Select the initial mobile and desktop direction
- [x] Document the planned technical architecture
- [ ] Validate the concept with potential users
- [ ] Confirm the product name and visual identity

## Phase 1 — Foundation

- [ ] Create the Flutter project structure
- [ ] Add adaptive navigation for mobile, tablet, and desktop
- [ ] Implement light, dark, and system-default themes
- [ ] Define reusable design tokens and components
- [ ] Establish local models and repository interfaces
- [ ] Add automated formatting, analysis, and tests

## Phase 2 — Personal organization MVP

- [ ] Today dashboard
- [ ] Tasks, subtasks, priorities, and deadlines
- [ ] Daily, weekly, monthly, and yearly calendar views
- [ ] Notes and checklists
- [ ] Local reminders
- [ ] Quick capture
- [ ] SQLite persistence
- [ ] Offline operation

## Phase 3 — Household organization

- [ ] Personal and household spaces
- [ ] Household members, roles, and permissions
- [ ] Shared shopping lists
- [ ] Household inventory
- [ ] Expiration tracking
- [ ] Recurring and rotating responsibilities
- [ ] Relevant household activity feed

## Phase 4 — Backend and synchronization

- [ ] FastAPI service foundation
- [ ] PostgreSQL schema and migrations
- [ ] Authentication and secure sessions
- [ ] Server-side authorization
- [ ] Offline-first synchronization
- [ ] Conflict handling and recovery
- [ ] Remote notifications
- [ ] File and image storage

## Phase 5 — Quality and release preparation

- [ ] Accessibility audit
- [ ] Security review
- [ ] Data export and account deletion
- [ ] Performance testing on representative devices
- [ ] Store assets and release documentation
- [ ] Privacy policy and terms
- [ ] Closed beta
- [ ] Public release

## Later possibilities

These are ideas, not commitments:

- Barcode scanning
- Receipt OCR
- Recipe and meal-planning connections
- Location-based reminders
- Widgets and lock-screen actions
- Voice capture
- Optional natural-language assistance
- Budget estimates for shopping lists
